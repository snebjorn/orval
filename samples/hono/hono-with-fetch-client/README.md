# hono-with-fetch-client

This is a sample app that uses `hono` as a backend and `Next.js` and `fetch` as clients for seamless schema-first development.

## hono-app

```
cd hono-app
pnpm install
pnpm dev
```

## next-app

```
cd next-app
pnpm install
pnpm dev
```
